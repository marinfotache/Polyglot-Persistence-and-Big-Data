SQL is undoubtedly the most important data processing language, not only for relational/SQL DBMSs, Hadoop (Hive, Impala, ...), and Spark (SparkSQL), but also for many NoSQL systems :-).
Despite its limitations - by default, R operates only in an allocated aread of computer's RAM -, R language is an excellent tool for collecting data from various sources in different formats, and subsequent activities related to data processing, curation, integration, exploration and analysis. 

In this section, main data processing features in `tidyverse` are presented by comparison with their SQL counterparts.
